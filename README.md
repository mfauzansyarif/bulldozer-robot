# Bulldozer Robot
ROS2 Jazzy mobile robot developed using ESP32 and differential drive architecture.

## Features
- ESP32 motor controller
- Differential drive robot
- Rotary encoder wheel odometry
- ROS2 odometry publisher
- TF transform publisher
- RViz2 visualization
- Goal-based autonomous navigation
- Manual teleoperation mode
- Serial communication between ROS2 and ESP32

## Technologies
- ROS2 Jazzy
- Python
- C++
- ESP32
- PlatformIO
- RViz2
- KiCad
